无家可归的卡尔加里人的身份证更换计划在资金枯竭后结束

Tue 07 June 2022

在资金枯竭后，无家可归的卡尔加里人的身份证更换计划结束 摄影：Gavin Young / Postmedia 一项帮助无家可归的卡尔加里人更换丢失的身份证件的计划在用尽联邦资金后已停止。2 点击此处查看我们团队的其他视频。尝试刷新您的浏览器，或在资金枯竭后结束无家可归的卡尔加里人的 ID 更换计划 返回视频 SORCe Calgary 的 ID 更换计划一直在利用联邦政府的 COVID-19 响应资金运作，资金现已枯竭。该计划每年的运营成本约为 100,000 美元，并帮助向卡尔加里人交付了约 1,300 份身份证件。其中包括出生证明、个人身份证和健康卡。高级主管凯文·布兰切特 (Kevin Blanchette) 说，当该组织收到联邦资金时，很快就注意到更换身份是一个关键问题。布兰切特说：“至关重要的是，人们拥有适当的身份证才能获得住房机会、收入支持和健康，这只是人们的三个主要需求。” SORCe 是一个多机构合作，将弱势的卡尔加里人与各种服务联系起来，主要与住房和健康有关。3 “确保他们拥有该 ID 对于他们继续访问支持他们获得可持续住房的服务的旅程非常重要。”布兰切特补充说，自上个月资助结束以来，SORCe 不得不拒绝 158 名寻求 ID 支持的人。 Chaz Smith 是 BeTheChangeYYC 的创始人，这是一个街头外展组织，定期与无人居住的卡尔加里人联系。他说，他所在机构的外展人员经常被问到他们可以去哪里获得身份证明。他说，SORCe 已终止的计划是一个方便无家可归的卡尔加里人获取所需文件的地方，这些文件可以开立银行账户或进入排毒床候补名单。 “它为访问此类节目创造了一个公平的场所，正如我们所知，ID 是您进入许多不同服务的切入点，”史密斯说。 “这只是一个非常有价值的程序。”4 布兰切特说，正在寻找维持服务的资金来源。他说，该计划在其生命周期内被证明是有效的，申请该计划的客户中有 85% 遵守并收到了他们的身份证。 “对我来说，这高度表明了这些人对确保更换身份证件的重要性，”他说。 “我们真的在评估并试图进一步建立案例以获得额外的资金，无论是公共资金还是私人资金，以便能够支持这个 ID 计划的发展。” jherring@postmedia.com 推特：@jasonfherring在您的社交网络中分享这篇文章最新的全国故事1 卡尔加里先驱报头条新闻 注册以接收来自 Postmedia Network Inc. 的一个部门 Calgary Herald 的每日头条新闻。电子邮件地址出现错误，请提供有效的电子邮件地址。注册 点击注册按钮，您同意接收来自 Postmedia Network Inc. 的上述时事通讯。您可以随时点击我们电子邮件底部的取消订阅链接取消订阅。后媒体网络公司| 365 Bloor Street East, Toronto, Ontario, M4W 3L4 | 416-383-2300 感谢您的注册！欢迎电子邮件正在发送中。如果您没有看到它，请检查您的垃圾文件夹。下一期卡尔加里先驱头条新闻将很快出现在您的收件箱中。我们在注册您时遇到了问题。请再试一次 文章来源 https://calgaryherald.com/news/local-news/homeless-id-replacement-program-shutters-after-funding-runs-dry#Echobox=1654387410
